# Банкомат (ATM)

Tag: atm

## Условие

Реализовать класс `Atm`, который хранит купюры фиксированных номиналов и выполняет операции пополнения и выдачи наличных. При ошибках операции должны завершаться выбросом исключения без изменения состояния банкомата.

## Номиналы

Фиксированный набор номиналов: `50, 100, 200, 500, 1000, 2000, 5000`.
Порядок выдачи: жадный алгоритм по убыванию номиналов.

## Публичный API

1. `Atm()` — создаёт пустой банкомат.
1. `void deposit(Map<Integer, Integer> banknotes)` — добавляет купюры (ключ — номинал, значение — количество).
1. `Map<Integer, Integer> withdraw(int amount)` — выдаёт сумму `amount` и возвращает разбиение по номиналам.
1. `int getBalance()` — возвращает текущий баланс.

## Правила ошибок

1. `InvalidDepositException` — если:
   - передано отрицательное или нулевое количество купюр;
   - номинал не входит в допустимый набор.
1. `InvalidAmountException` — если `amount <= 0`.
1. `InsufficientFundsException` — если `amount > getBalance()`.
1. `CannotDispenseException` — если сумма не набирается доступными купюрами.

Во всех случаях при исключении состояние банкомата не изменяется.

## Примеры

**Пополнение и баланс:**
- Изначально банкомат пуст, баланс 0.
- Пополнение: `{100=10, 500=5}` → баланс `3500`.
- После выдачи `700` (например, `1×500 + 2×100`) баланс `2800`.

**Ошибки:**
- Запрос `10_000` при балансе `3000` → `InsufficientFundsException`.
- Запрос `375` → `CannotDispenseException`.
- Вызов `withdraw(-100)` → `InvalidAmountException`.
